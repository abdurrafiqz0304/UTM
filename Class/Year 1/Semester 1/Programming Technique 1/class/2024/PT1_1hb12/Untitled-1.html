Rectangle Area Calculator Explanation

Program Overview

The program is a simple console-based application that calculates the area and perimeter of a rectangle. It repeatedly prompts the user to enter the rectangle's dimensions and displays the results until the user decides to stop.

Key Components

  • getPositiveInput Function:
    • This function prompts the user for input and ensures that the input is a positive number.
    • It takes a std::string as a parameter, which is used to display the prompt message.
    • The function uses a do-while loop to repeatedly ask for input until a valid positive number is entered.
  • displayResults Function:
    • This function calculates the area and perimeter of the rectangle using the provided length and width.
    • It then displays the length, width, area, and perimeter in a formatted manner.
    • This function encapsulates the logic for displaying results, making the main function cleaner and more focused.
  • main Function:
    • The main function serves as the entry point of the program.
    • It displays a welcome message and enters a loop where it repeatedly asks the user for the rectangle's dimensions.
    • It calls getPositiveInput twice to get valid length and width values.
    • It then calls displayResults to show the calculated area and perimeter.
    • The loop continues based on the user's choice, which is checked using a do-while loop. The loop exits if the user enters 'n' or 'N'.

Modern C++ Practices

  • Use of std::string: The program uses std::string for handling prompt messages, which is a modern C++ practice for handling strings.
  • Encapsulation: The program encapsulates input validation and result display in separate functions, improving code readability and maintainability.
  • Consistent Formatting: The code is formatted consistently, with clear separation of logic and comments explaining each part.

Line-by-Line Explanation

#include <iostream>

double getPositiveInput(const std::string& prompt) {
    double value;
    do {
        std::cout << prompt;
        std::cin >> value;
        if (value <= 0.0) {
            std::cout << "Please enter a positive number.\n";
        }
    } while (value <= 0.0);
    return value;
}

void displayResults(double length, double width) {
    double area = length * width;
    double perimeter = 2 * (length + width);
    std::cout << "\nRectangle's Length: " << length << "m"
              << "\nRectangle's Width : " << width << "m"
              << "\nRectangle's Area  : " << area << " sq.m"
              << "\nRectangle's Perimeter: " << perimeter << "m\n";
}

int main() {
    std::cout << "This Is Rectangle Area Calculator\n";
    char choice;
    do {
        double length = getPositiveInput("\nEnter Rectangle's Length (positive number): ");
        double width = getPositiveInput("Enter Rectangle's Width (positive number): ");
        
        displayResults(length, width);

        std::cout << "\nDo you want to continue? [y/n] ";
        std::cin >> choice;
    } while (choice == 'y' || choice == 'Y');

    return 0;
}

This program is structured to ensure input validation, clear output, and user interaction in a loop, allowing repeated calculations until the user decides to stop.
